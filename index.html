Installation Instructions

  1. Open downloads.
  2. Click download file.
  3. Accept "download from unknown source" (it'll be detected as a possible virus).
  4. Place the downloaded file in a secure place (Documents, Desktop, etc).
  5. Go to Chrome's extensions page, enable Developer Mode, and click "Load unpacked extension".
  6. Click the extension icon.
  7. Add API key and secret.
  8. Open WhatsApp.
